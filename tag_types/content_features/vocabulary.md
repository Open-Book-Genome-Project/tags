# Content Features — Vocabulary

An open/managed list. Additions via PR with brief justification. See [CONTRIBUTING.md](../CONTRIBUTING.md).

Tag format: `content_feature:term` (e.g., `content_feature:maps`)

---

## Paratextual Features

Structural material surrounding or supplementing the main text.

| Tag | Definition |
|---|---|
| Bibliography / References | A list of sources cited or consulted by the author |
| Charts and graphs | Data visualizations included as part of the work |
| Discussion questions | Questions for group discussion, typically at the work's end |
| Footnotes / Endnotes | Supplementary notes at the bottom of pages or end of sections |
| Glossary | A defined list of specialized terms used in the work |
| Illustrations (black and white) | Black and white drawings, engravings, or artwork within the text |
| Illustrations (color) | Full-color artwork, photographs, or plates within the text |
| Index | Alphabetical index of names/subjects with page references |
| Maps | Geographic or world maps included as part of the work |
| Problem sets / Exercises | Practice problems or activities for the reader to complete |
| Table of contents | Structured listing of chapters/sections with page numbers |

---

## Narrative / Structural Features

Formal properties of how the text itself is constructed.

| Tag | Definition |
|---|---|
| Epistolary structure | Narrative constructed from letters, diary entries, or found documents |
| Frame narrative | A story-within-a-story: an outer narrative containing the central tale |
| Multiple narrators | Two or more distinct narrative voices |
| Non-linear chronology | Chapters/sections are not arranged in chronological order |
| Second-person narration | Narrated in the second person ("you do X") |
| Verse / Poetic prose | Written in verse, or in prose with significant lyrical structure |
