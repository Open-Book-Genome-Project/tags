# subgenres

**Definition:** Specific structural frameworks, world-states, or narrative logics that operate *within* a genre. They answer the question "how is this story structured?" or "what kind of world does this take place in?"

---

## What is a Subgenre?

A subgenre is not a softer version of a genre — it is a **technical descriptor** of a story's structural mechanics or the state of the world it depicts.

Subgenres narrow a genre without replacing it. A work tagged `Cyberpunk` is still `Sci-Fi`; `Locked Room` still needs `Mystery`; `Gothic` still needs `Horror` (or `Romance`, depending on the work).

---

## Core Rules

**Do use subgenres to capture:**
- The **structural logic** of the plot (Locked Room, Picaresque, Coming of Age)
- The **state of the world** as a defining structural constraint (Dystopian, Utopian, Post-Apocalyptic)
- The **technological or aesthetic framework** that defines the setting's mechanics (Cyberpunk, Steampunk, Biopunk)
- The **scale** of the narrative (Epic, Space Opera)

**Do NOT put in subgenres:**
- Setting mashups — "Historical Fantasy" is not a subgenre; use `Historical` (genre) + `Fantasy` (genre)
- Simple subjects — "Time Travel" describes a plot device, not a structure; use `things` or `main_topics`
- Emotions — those belong in `moods`
- Broad emotional registers — those belong in `genres`

**The Framework Rule:** A subgenre must describe the *how* of the plot, not the *what*. It is the architecture of the story, not its furniture.

**The No-Mashup Rule:** Never create a subgenre by combining two genres (e.g., no "Historical Fantasy", no "Romantic Thriller"). Instead, apply both genres.

---

## Relationship to Genres

Every subgenre implies one or more parent genres. Some examples:

| Subgenre | Typical Parent Genre(s) |
|---|---|
| Cyberpunk | Sci-Fi |
| Gothic | Horror, Romance, Literary |
| Locked Room | Mystery, Crime |
| Coming of Age | Drama, Literary, Romance, Fantasy |
| Space Opera | Sci-Fi |
| True Crime | Crime |
| Family Saga | Drama, Historical |
| Dystopian | Sci-Fi, Fantasy |

A subgenre can be applied without explicitly stating the parent genre if the parent is strongly implied, but best practice is to include both.

## Proposing a New Subgenre

See [CONTRIBUTING.md](../CONTRIBUTING.md). A new subgenre must:

1. Describe the structural logic, world-state, or narrative framework of the story
2. Not be expressible as a setting mashup of two existing genres
3. Not be a topic, prop device, or theme
4. Modify a genre rather than replace it
