# Content Formats — Controlled Vocabulary

A managed list. To propose additions or changes, see [CONTRIBUTING.md](../CONTRIBUTING.md).

Tag format: `content_format:term` (e.g., `content_format:novel`)

---

| Tag | Definition |
|---|---|
| Almanac | Annual reference compiling statistics, facts, and forecasts organized by date or category |
| Anthology | A curated collection of works (stories, poems, essays) by multiple authors |
| Artbook | A work whose primary content is visual art, with minimal or supplementary text |
| Atlas | A collection of maps, often with accompanying geographic or thematic data |
| Autobiography | A first-person account of the author's own life, written by the subject |
| Biography | An account of a person's life written by someone other than the subject |
| Chronology | A work organized by dates and events, tracing a timeline |
| Diary | Written in journal or diary form — dated entries presented as contemporaneous |
| Dictionary | A reference work organized alphabetically, defining terms |
| Encyclopedia | A comprehensive reference work covering a broad range of subjects |
| Essays | A collection of short non-fiction prose pieces on distinct topics |
| Graphic Novel | A narrative told primarily through sequential art (comics format); longer form |
| Letters | A collection of correspondence, organized around an epistolary exchange |
| Lexicon | A specialized vocabulary or word list for a particular subject domain |
| Manga | Sequential art in Japanese comics format; typically serialized and read right-to-left |
| Memoir | A first-person narrative focused on a specific period, theme, or set of experiences in the author's life |
| Novel | Long-form prose fiction (or narrative nonfiction); typically 40,000+ words |
| Novella | Mid-length prose narrative; typically 17,500–40,000 words |
| Photography | A work whose primary content is photographic images |
| Recipes | A collection of cooking or craft instructions organized around preparation |
| Reference | A work designed for lookup rather than linear reading |
| Script / Play | Formatted as dramatic dialogue and stage directions; written for performance |
| Short Story | A brief prose narrative; typically under 7,500 words; often collected in anthologies |
| Textbook | A structured educational work designed for formal instruction in a subject |
